WEBSITE

Browser:
it a software that renders and displays a webpage.it is use to interact and view content of a webpage.They are application software such as opera,firefox,mozilla,google,etc. A browser is also a program that allows you to interact and view contents of the Web (Application programs dedicated for web browsing)
Internet:
it is the network of massive networks.it is combination of different computer networks that have website.it is also a massive network of networks. It connects millions of computers together globally, forming a network in which any computer can communicate with any other computer as long as they are both connected to the Internet.
Client:
This is any device that have both internet connectivity and browser. Devices (e.g. laptop, tablet, mobile phones etc.) from which browsers are used are referred to clients.
Server:
It is any dedicated machine running 24/7 which is used to give information to the client.computers can be called a server because it is serving the website when connected to the internet, by spining website for information.When a computer is connected to the internet and spinning out a website, we call the computer a server because it is serving the website.
WWW:
it is an information sharing model built on the internet.it is used in giving out information within the medium of the internet.The World Wide Web, or simply Web, is a way of accessing information over the medium of the Internet. It is an information-sharing model that is built on top of the Internet (A bunch of connected computers that have websites).
